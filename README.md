# test-new-issus

Repo sandbox pour valider visuellement les **issue templates** avant de les déployer sur `ROBOT-COUPE/VP-nextgen-sf`.

## Comment tester

1. Aller sur l'onglet [Issues](../../issues)
2. Cliquer sur **New Issue**
3. Choisir un des templates (Bug report / Feature request / Task)
4. Remplir le formulaire et soumettre
5. Vérifier le rendu de l'issue créée

Quand le rendu est satisfaisant, on copie les fichiers `.github/ISSUE_TEMPLATE/*.yml` sur le repo cible.
